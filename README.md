# Mecanica Automotriz Web

Professional web system for an automotive repair shop, built with Angular, TypeScript, Supabase and Netlify.

## Project Objective

This project aims to provide a modern, stable and scalable web platform for managing an automotive repair shop. The system will support public information pages, appointment requests, customer management, vehicle records, work orders, spare parts and internal reports.

## Technology Stack

- Angular
- TypeScript
- SCSS
- Supabase PostgreSQL
- Supabase Auth
- Supabase Storage
- Netlify

## Planned Modules

- Public website
- Authentication
- Dashboard
- Customers
- Vehicles
- Appointments
- Work orders
- Spare parts
- Reports
- User roles and permissions

## Architecture

The project follows a modular frontend architecture using Angular feature modules and a clear separation between core services, shared components and business features.

```txt
src/
 └── app/
     ├── core/
     ├── shared/
     ├── features/
     └── app.routes.ts
```
